# 📦 Perspective Box Trainer

Тренажёр для отработки перспективы при рисовании кубов — прямо в браузере, без установки. Полезен художникам, иллюстраторам и всем, кто учится рисовать в перспективе.

**🔗 Демо:** https://ggop445637.github.io/perspective-box-trainer/

## ✨ Возможности

- **Режим «Практика»** — куб(ы) в случайной перспективе появляются на экране на заданное время, после чего сцена обновляется. Отлично подходит для тренировки на скорость.
- **Режим «Нарисуй и проверь»** — куб показывается на несколько секунд для запоминания, затем исчезает. Вы рисуете его по памяти мышью или пальцем, а затем сверяете свой рисунок с правильным вариантом (полупрозрачным наложением).
- **3 уровня сложности перспективы** — от простой фронтальной до динамичной с произвольными углами.
- **От 1 до 3 кубов** одновременно, с одинаковой или разной перспективой для каждого.
- **Таймер** с настраиваемой длительностью (5–30 секунд) или без ограничения по времени.
- **Свободное вращение камеры** — перетаскивание для орбиты вокруг сцены, колесо мыши / щипок двумя пальцами для зума, двойной клик/тап для сброса вида.
- **Статистика и серии (streak)** — счётчик нарисованных кубов за сегодня, всего и серия дней подряд с практикой. Сохраняется локально в браузере.
- **Тёмная/светлая тема** + настройка цветов кубов и акцента.
- **Полноэкранный режим** и горячие клавиши.
- **Двуязычный интерфейс** — русский и английский.
- Полностью офлайн-совместимо: один HTML-файл, без серверной части и внешних зависимостей.

## ⌨️ Горячие клавиши

| Клавиша | Действие |
|---|---|
| `Space` / `N` | Новая поза / новая задача |
| `F` | Полноэкранный режим |
| `T` | Пауза таймера |
| `R` | Сброс вращения и зума камеры |
| `M` | Скрыть / показать меню |

## 🚀 Использование

Просто откройте `index.html` в браузере — никакой установки или сборки не требуется.

### Запуск через GitHub Pages

1. Settings → Pages → Source: `main` / `root`.
2. Через пару минут сайт будет доступен по адресу вида `https://username.github.io/repo-name/`.
