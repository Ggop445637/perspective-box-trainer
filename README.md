# 📦 Perspective Box Trainer

**Бесплатный онлайн-тренажёр перспективы для художников и иллюстраторов.**

Рисуйте кубы и цилиндры в правильной перспективе, тренируйте глаз на скорость или проверяйте себя по памяти — прямо в браузере, без установки.

🔗 **[Открыть тренажёр](https://ggop445637.github.io/perspective-box-trainer/)**

---

## ✨ Возможности

### 🎯 Два режима тренировки

- **Практика** — генератор бесконечно случайных кубов и цилиндров в разных ракурсах. Смотрите на фигуру, рисуйте на бумаге или планшете рядом, нажимайте «Генерировать» (или просто `Space`) для новой.
- **Тест памяти** — фигура показывается на несколько секунд, затем скрывается. Нужно нарисовать её проекцию по памяти прямо на экране (мышью или пальцем) и сравнить со скрытым эталоном.

### 📐 Видимость точек схода и линий перспективы

Опциональная (включаемая по кнопке) подсказка прямо на фигуре:

- Показывает **реальные точки схода** перспективы и направляющие линии, построенные точно по математике текущей проекции — линии всегда проходят ровно по рёбрам фигуры, без визуальных допущений.
- **Количество точек схода подстраивается под угол обзора само**: фронтальный ракурс показывает 1 точку (одноточечная перспектива), ракурс «с угла» — 2 точки, динамичный наклонённый ракурс — 3 точки. Никаких лишних точек, уезжающих за пределы экрана.
- Работает одновременно в обоих режимах — «Практика» и «Тест памяти».

### 🔷 Фигуры и сложность ракурса

- **Куб** и **Цилиндр** — переключаются одной кнопкой.
- Три уровня сложности ракурса:
  - **Простой** — близкий к фронтальному виду, для новичков.
  - **Угловой** — заметный поворот, классическая учебная перспектива.
  - **Динамика** — резкие, непредсказуемые ракурсы для продвинутой практики.

### 🧊 Несколько объектов одновременно

- От 1 до 3 фигур на сцене одновременно.
- Режим **«Общая перспектива»** — все объекты развёрнуты в одном направлении (как в реальной сцене с одним источником перспективы).
- Режим **«Разная перспектива»** — у каждого объекта собственный случайный ракурс, для тренировки на нескольких независимых формах.

### 🖱️ Свободное вращение камеры

- Включаемый режим орбитальной камеры: вращайте сцену мышью (десктоп) или пальцем (мобильные/планшеты), приближайте и отдаляйте колесом мыши или щипком (pinch-to-zoom).
- Двойной клик/тап — мгновенный плавный сброс камеры в исходное положение.

### ⏱️ Таймеры и авто-режимы

- **Практика**: настраиваемый интервал автогенерации новой фигуры — 5 / 10 / 15 / 30 секунд, либо полностью бесконечный режим без таймера. Таймер можно ставить на паузу кликом по нему.
- **Тест памяти**: 
  - Настраиваемое время показа фигуры перед скрытием (2 / 3 / 5 секунд).
  - После проверки автоматически даётся 5 секунд на просмотр правильного ответа перед тем, как появится новая фигура.
  - **Авто-проверка** (опционально) — если включена, тренажёр сам нажимает «Проверить» по истечении заданного времени на рисование (5 / 10 / 15 / 30 секунд), превращая весь цикл «показ → рисование → проверка → следующая фигура» в полностью автоматический поток, без необходимости нажимать кнопки руками.

### 📊 Точность, статистика и прогресс

- **Оценка точности** в режиме «Тест памяти» — нарисованный контур сравнивается с эталоном, результат показывается в процентах и оценке (от «Повторить» до «💎 Мастер»).
- **Счётчик отрисованных сессий**, статистика **за сегодня** и **всего**.
- **Серия дней подряд** (streak) с индикатором огня, растущим по мере серии.
- **Система уровней** — каждые 100 нарисованных фигур = новый уровень, с медалями (🥉 → 🥈 → 🥇 → 🏆).
- Вся статистика сохраняется локально в браузере и не пропадает между сессиями.

### 🎨 Внешний вид

- **Тёмная и светлая тема** — переключение одной кнопкой.
- **Полностью настраиваемая палитра**: акцентный цвет интерфейса, цвета до трёх объектов на сцене, отдельный цвет для эталона в тесте памяти. Настройки сохраняются автоматически.
- Чистый, минималистичный glass-UI интерфейс, который можно свернуть в один клик, чтобы не мешал во время рисования на отдельном экране/планшете.

### 🌍 Языки

- Русский и английский интерфейс, переключение в один клик.

### 📱 Адаптивность

- Полностью адаптирован под мобильные устройства и планшеты: интерфейс, размер фигур и элементы управления подстраиваются под размер экрана.
- Поддержка касаний: рисование пальцем, вращение камеры жестами, pinch-to-zoom.
- Режим полного экрана для максимально чистого холста.

### ⌨️ Горячие клавиши (десктоп)

| Клавиша | Действие |
|---|---|
| `Space` / `N` | Новая фигура / новая генерация |
| `F` | Полноэкранный режим |
| `T` | Пауза / снятие с паузы таймера |
| `R` | Сброс ракурса камеры |
| Двойной клик | Сброс ракурса камеры (при свободном вращении) |

## 🚀 Запуск

Тренажёр — это один самодостаточный HTML-файл, не требующий сборки, серверной части или установки зависимостей.

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

Просто откройте `index.html` (или соответствующий `.html` файл) в любом современном браузере — Chrome, Firefox, Safari, Edge.

### Деплой на GitHub Pages

1. В настройках репозитория откройте **Settings → Pages**.
2. Укажите ветку (например, `main`) и папку `/ (root)`.
3. Сайт будет доступен по адресу `https://your-username.github.io/your-repo-name/`.

## 🛠️ Технологии

- **Чистый HTML / CSS / JavaScript** — без фреймворков, без сборщиков, без зависимостей.
- Рендеринг через **2D Canvas API** с собственной реализацией перспективной проекции (ручная орбитальная камера, точная математика точек схода, скрытие невидимых рёбер).
- Хранение настроек и статистики через `localStorage` — всё работает офлайн после первой загрузки.
